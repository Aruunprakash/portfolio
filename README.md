# Arun Prakash - Personal Portfolio

A sleek, responsive, and interactive personal portfolio showcasing my journey, projects, and skills as an AI/ML and Data Science enthusiast. Designed to highlight technical proficiencies and continuous learning in intelligent systems and software development.

[View Portfolio Live](https://aruunprakash.github.io/portfolio)

## Overview

This repository contains the source code for my personal portfolio. It is built from the ground up using modern web standards to ensure a clean, performant, and engaging user experience. The design features dynamic animations, a premium aesthetic, and intuitive navigation.

## Key Features

- **Responsive Design**: Adapts seamlessly to desktop, tablet, and mobile devices.
- **Dynamic Animations**: Incorporates advanced CSS and JavaScript animations, including hero grid drifts, scrolling marquees, section title reveals, and 3D tilt effects for project cards.
- **Project Showcase**: A dedicated section to display AI/ML, Data Science, and development projects.
- **Skills & Statistics**: Visual representation of technical proficiencies and key milestones.
- **Interactive UI Elements**: Features like a vertical dot navigator, dynamic contact CTA, and live local time integration to enhance user engagement.
- **Accessibility & Performance**: Developed keeping reduced-motion preferences and optimal performance in mind.

## Technology Stack

- **HTML5**: Semantic structure and content.
- **CSS3**: Vanilla CSS tailored for custom styling, fluid typography, and complex keyframe animations (no external bloated frameworks).
- **JavaScript (ES6+)**: Core logic for scroll interactions, intersection observers, and interactive UI components.
- **Git & GitHub Pages**: Version control and seamless continuous deployment.

## Project Structure

- `index.html`: The main structural entry point of the website.
- `style.css`: Contains all custom properties, design tokens, responsive queries, and animations.
- `script.js`: Manages interactivity, scroll observers, dynamic counters, and DOM manipulations.

## Local Setup

To run this project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/Aruunprakash/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your preferred web browser. No local development server or build process is strictly required, though a tool like Live Server (VS Code) is recommended for a better development experience.
